Simple Imp

This is a simple imperative language with numbers, Booleans, and records. Additionally, one extension adds a new analysis to determine if a program can leak private information.

  • Host language: Simple imperative language with numbers and records
  • Security extension: Adds an analysis to determine if a program can leak private information
  • List extension: Adds syntax for list expressions
  • Repeat-while extension: Adds syntax for repeat-while loops
